HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Metadata is data read by the browser. It is located in the head element
  • Headers are contained in the body. They can be tagged with h1 through h6 depending on the emphasis needed.
  • The p tag is used to represent a block of text or paragraph on the page
  • Lists can be ordered or unordered and are tagged respectively with ol or ul
  • List items are tagged with the element li
  • The element br created a line of empty space or a "break"
  • The img element contains info about displayed images on the page. The most important info is the src element, which defines the location of the image.
  • The a (anchor) element creates links to the same or other webpages. An href attribute defines the url linked.
  • HTML functions as the framework for your webpage. Its like the frame and foundation of a house.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Inine CSS: apply to a specific HTML tag. less flexible as they must all be set individually.
  • Internal style sheet: added to the head element by a style element. Sets rules for an entire page for specific elements.
  • External CSS style sheet: allows developers to keep all CSS rules stored in a different file, making design changes easier.
  • Colors may be notated semantically or by hex code
  • Class selectors may be used to assign a specific HTML element to mutliple different elements.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: returns you to the main branch
  • cd: change directories
  • pwd: displays current working direcotry
  • ls: displays a list of files in current directory
  • mkdir: creates a new directory
  • touch: creates a file inside a directory
  • open (mac) or start (windows): opens a file

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script
  • A function is a set of instructions that tells a computer how to perform a certaint task.
  • For a function to execute, you must call the function after defining it.
  • An array is used to store a group of related data in a single variable. They cna hold multiple types of data.
  • The location of an item in an array is identified by an index. This is a number starting at 0 to the end of the array.
  • A for loop can be used to execute a line of code across an entire array of variables.
  • A for loop has 3 statements: One for its starting point, A second for the condition to run, and a final statement to allow the arrray to iterate over each item.
  • Most Javascript coding is done using conditional logic, like for and if statements.
